# ♟ Easy Chess (Python + Tkinter)

Simple chess game with graphical interface and AI opponent.

## 🎮 Features

- Interactive chessboard
- Highlighting of possible moves
- Capture highlighting
- Player vs Computer mode
- Simple AI opponent
- Reset game button

## 🧠 Technologies

- Python
- Tkinter
- Dataclasses
- Basic AI logic
